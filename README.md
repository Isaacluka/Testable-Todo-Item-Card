# Testable-Todo-Item-Card
This is a simple responsive Todo/ Task Card component.

## Live Demo
https://isaacluka.github.io/Testable-Todo-Item-Card/

## Features
- Responsive card component
- Toggleable checkbox to indicate completion of task
- JS populated div that indicateds when the task is pending and when it is done.


## Tech Stack
- HTML
- CSS
- JavaScript
